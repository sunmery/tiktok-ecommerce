## 微服务端口分配
目前每个微服务都有自己的 HTTP 和 gRPC 端口, 根据Kubernetes NodePort Service的端口(默认范围：30000-32767)分配, 从 30000 端口开始, 分配如下:
- auth: 30001,30002
- user: 30003,30004
- product: 30005,30006
- cart: 30007,30008
- order: 30009,30010
- checkout: 30011,30012
- payment: 30013,30014
- category: 30015,30016
- assistant: 30017,30018
- merchants: 30019,30020
- admin: 30021, 30022
- comment: 30023, 30024

## 启动基础设施

make infra
